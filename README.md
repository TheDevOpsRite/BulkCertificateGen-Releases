Bulk Certificate Generator - The DevOps Rite
# Features
- Read your Excel file
- Allows you to upload certificate template 
- Allows you to select the position
- Preview Certificate button avilable
- Ability to generate bulk certificate

# Working
- Download the setup.
- Click Keep via three dots option. (Browser can show warning as code is not signed)
- Double click to open the installer and click Yes. (Click Yes to allow the installation)
- Provide the custom path where You want to install the setup.
- Check the Launch option and click finish to Launch the application.
- Provide your Certificate template and Excel file.
- Adjust the position of the text accordingly and click preview.
- Find the Generated Certificates in output folder where you installed the Application.
- Click Bulk Generate to generate all the certificates according to your excel file.
